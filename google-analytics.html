Google Analytics Setup

Maltas Esport Community - maltasesport.my.id

1. Google Analytics 4 Code

Tambahkan kode berikut sebelum closing </head> tag:

<!-- Google Analytics 4 --> <script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'GA_MEASUREMENT_ID'); </script>

Catatan: Ganti GA_MEASUREMENT_ID dengan ID Google Analytics Anda

2. Google Search Console

Verifikasi website di Google Search Console menggunakan:

  • HTML file upload (google-site-verification.html)
  • Meta tag verification
  • Google Analytics verification

3. Sitemap Submission

Submit sitemap.xml ke Google Search Console:

https://maltasesport.my.id/sitemap.xml

4. Bing Webmaster Tools

Verifikasi website di Bing menggunakan BingSiteAuth.xml
